1. Projects Tittle: Bank Application Back-End structure.
2. Project Description: Creation and design of Back-end structure. The system have 4 types of Account " StudentChecking", "Cheking", "Savings" and "CreditCard".
   The system also have 3 types of User "AccountHolders", "Admins" and "Third-Party".
   Admins can creat new accounts.
   The system should be able to apply correctly all the fees and interest rates described.
3. Technologies: The project has been developed using JAVA 17 language, SpringBoot framework, JUNIT for testing.
4. Project Status: There are some functionalities and tests that are not finished yet as the project is still ben developed.
